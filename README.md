# AC 2-3前端 驗收 Express應用

### 功能說明
於本機建立前端頁面，可點擊切換不同頁


### 環境建置
- Node.js 14.16.0
- Express 4.16.4
- Express-Handlebars 3.0.0



### 安裝&執行
1. 請先確認本機有安裝Nodde.js跟npm
2. 下載專案
3. 從終端機移動到專案資料夾後輸入

   ```bash
   npm install
   ```

4. 安裝完畢後，繼續輸入

   ```bash
   npm run start
   ```

5. 看到下方訊息代表順利執行


   ```bash
   Listening on http://localhost:3000
   ```
